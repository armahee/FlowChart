4 The immediate is fetched
5 Barrel shifter is used according to the offset in the instruction.
6 Rn and immediate are finally subjected to ALU which computes the value by performing arithmetic and logical operations required.
7 Final value is stored in Rd in Register bank
8 Rd is populated with the value to be stored. It is read on Bus B.
9 It is stored in data out register.
10 Rn is populated with the value of address stored and the PC is returned
